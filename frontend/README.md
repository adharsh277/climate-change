# Frontend

This folder contains all the frontend code for the Climate Change - Rainwater Harvesting project.

## 📂 Structure

```
frontend/
├── assets/
│   ├── images/      # 77 image files
│   └── videos/      # Video files for the project
├── css/             # All stylesheets
├── js/              # JavaScript files
└── *.html           # HTML pages
```

## 📄 Pages

- `index.html` - Homepage
- `commercial.html` - Commercial information
- `contact.html` - Contact page
- `development.html` - Development page
- `eco.html` - Eco-friendly solutions
- `method.html` - Methods page
- `news2.html` - News/Updates
- `product.html` - Products page
- `residential.html` - Residential information

## 🎨 Assets

- **Images:** 77 images organized in `/assets/images/`
- **Videos:** Educational videos in `/assets/videos/`

## 🚀 Getting Started

Simply open `index.html` in a web browser to view the website locally, or deploy the frontend folder to any static hosting service.
